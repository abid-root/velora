# Velora Sourcing International Static Website

This is a complete static website package with HTML, CSS, JavaScript, images, brand assets, SEO files, legal pages, blog pages, production pages, service pages, contact pages, FAQ, 404, robots.txt, sitemap.xml, and manifest.webmanifest.

## How to run locally
Open `index.html` directly or use VS Code Live Server.

## Important before launch
- Replace placeholder phone numbers, office addresses, and email addresses.
- Show only verified compliance certificates.
- Connect the contact form to a real form service later if needed.
- Update `robots.txt` and `sitemap.xml` with the final domain.
